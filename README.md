# Mountain-Searching-Engine

```
此搜尋引擎主要供外國人使用，輸入台灣山的英文名稱，並且由使用者給予三個關鍵字來加權計算得到網頁爬蟲後的分數，若該關鍵字在網頁出現越多次
，則分數會越高，我們取分數前十高的網頁做為搜尋結果。
```
