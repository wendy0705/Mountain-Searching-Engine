# Mountain-Searching-Engine

```
此搜尋引擎主要供外國人使用，輸入台灣山的英文名稱，由使用者給予三個關鍵字給予最適配的網頁搜尋結果。
方法為透過關鍵字進行網頁爬蟲後進行加權，因此該關鍵字在網頁出現越多次，分數會越高，我們取分數前十高
的網頁做為搜尋結果。
```
